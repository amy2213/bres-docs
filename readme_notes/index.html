    Phaistos Disc: BRES Classification Dataset

    This repository provides a complete breakdown of the Phaistos Disc's glyphs as interpreted through the Binary Ritual Encoding System (BRES).

    üîç Dataset Features

    • Full glyph index with symbols, sector placement, and cultural interpretation
    • Cleaned BRES Phase assignments per glyph
    • Full BRES Sector Phase Sequences
    • Sector glyph orders
    • Interpretive context and ritual symbolism
